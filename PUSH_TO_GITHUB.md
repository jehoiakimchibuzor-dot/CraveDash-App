# 🚀 FINAL STEP: Pushing CraveDash-App to GitHub

Follow these steps exactly. Copy one box at a time, paste it into the terminal, and press **Enter**.

### Step 1: Add all files
This prepares all your project files for saving.
```sh
git add .
```

### Step 2: Create the "Save Point" (Commit)
This officially saves the current version of your code on your computer.
```sh
git commit -m "Initial commit: CraveDash-App Premium MVP with AI MAX"
```

### Step 3: Set your branch name
```sh
git branch -M main
```

### Step 4: Link your computer to GitHub
**⚠️ IMPORTANT:** Replace `YOUR_GITHUB_USERNAME` with your actual name on GitHub (e.g., `jehoiakim-dev`).
```sh
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/CraveDash-App.git
```

### Step 5: Push to the Cloud
This sends your code to GitHub. You might be asked to log in in a browser window.
```sh
git push -u origin main
```
