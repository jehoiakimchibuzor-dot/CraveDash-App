# CraveDash: A Premium AI-Powered Conversational Food Ecosystem

### Project Overview
CraveDash is a next-generation food delivery application designed for the Nigerian market, built with a "Security-First" and "User-Centric" philosophy. While traditional apps rely on rigid menus and robotic support, CraveDash leverages cutting-edge Large Language Models (LLMs) to bridge the gap between digital commerce and human-like interaction.

### The "Genius" Core: MAX (AI Concierge)
At the heart of the ecosystem is **MAX**, an elite AI concierge powered by **Google Gemini 1.5 Flash**. MAX is not a standard chatbot; he is a street-aware, emotionally intelligent food expert. Using advanced prompt engineering, MAX is designed to:

*   **Detect Customer "Vibes"**: Automatically adjusts his tone based on user energy—handling everything from casual slang to professional business inquiries with saint-like patience and sharp wit.
*   **Conversational Commerce**: MAX understands subtext and intent, guiding users through the menu, suggesting meals based on cravings, and handling support issues without ever using robotic "I don't understand" templates.
*   **Hyper-Local Knowledge**: Deeply integrated with Nigerian expressions and menu specificities, ensuring a premium, tailored experience.

### Technical Architecture & Security
CraveDash demonstrates industry-standard Android development practices:

*   **Modern UI/UX**: 100% **Jetpack Compose** architecture for a fluid, reactive interface. High-contrast themes and precise window inset management ensure a seamless experience, including "WhatsApp-standard" keyboard behavior.
*   **Secure Infrastructure**: Implements a robust secret management system using the **Secrets Gradle Plugin** and **BuildConfig**. API keys are strictly isolated in `local.properties` and ignored by Git, ensuring a zero-leak footprint for production environments.
*   **Safe Navigation**: Uses a type-safe Compose Navigation strategy with encoded URL parameters for complex data passing between screens.
*   **Feature Rich**: Includes a high-performance Dashboard, Categorized Menu, Realistic Food Details, Order History, and a real-time Progress Stepper for order tracking.

### The Vision
CraveDash serves as a proof-of-concept for the future of "Conversational Commerce"—where technology feels less like a tool and more like a smart friend who knows exactly what you want, "sharp sharp."
